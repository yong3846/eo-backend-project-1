# AI Portal Project

## 프로젝트 소개 (Korean)

이 프로젝트는 **[ESTSOFT] 백엔드 개발자 양성과정 14회차**의  
**첫 번째 프로젝트**입니다.

간단한 **AI 포털 사이트의 화면** 을 설계하는 것을 목표로 하며,  
AI 서비스들을 한 곳에서 접근할 수 있는 포털 형태의 웹 페이지를 구상했습니다.

> 본 README는 프로젝트 진행 상황에 따라 지속적으로 업데이트될 예정입니다.

## Project Overview (English)

This project is the **first project** of  
**[ESTSOFT] Backend Developer Training Program (14th batch)**.

The goal of this project is to design a **simple AI portal website UI**,  
providing a conceptual layout for accessing multiple AI services from a single portal.

> This README will be continuously updated as the project progresses.
